# PRODIGY_WD_05 — Weather App
**Task 5** for Prodigy InfoTech Web Development Internship (December 2025)

## Live Demo
https://krishnapandey2006.github.io/PRODIGY_WD_05/

## Overview
A simple weather app that allows users to search for a city and view the current temperature, wind speed, humidity (approx), and a short condition summary with an emoji.

## Features
- City name search
- Geocoding using Open-Meteo's free API
- Current temperature and wind speed
- Humidity approximation from hourly data
- Weather condition text + emoji
- Responsive, clean UI

## Tech
- HTML5, CSS3, Vanilla JavaScript
- Open-Meteo API (no API key required)

## How to run locally
1. `git clone https://github.com/krishnapandey2006/PRODIGY_WD_05.git`
2. `cd PRODIGY_WD_05`
3. Open `index.html` in your browser.
